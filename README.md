# 钙钛矿称量计算器

一个离线可用、也适合部署到 GitHub Pages 的单页称量计算器。

## 功能

- 根据 FA / MA / Cs / Br 比例、Pb 摩尔浓度和溶液体积计算 FAI、MAI、CsI、PbI2、PbBr2、RbCl 的称量质量。
- 分子量内置且可修改。
- 可以保存本次填写记录，记录会带时间戳保存在当前浏览器中。
- 可以导出保存记录为 CSV。

## 使用

直接打开 `perovskite-weighing-calculator.html`，或部署后访问 GitHub Pages 首页。

记录保存在浏览器 `localStorage` 中。不同设备或不同浏览器之间不会自动同步。
